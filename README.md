# K-Hub Portfolio | Full Stack Developer & Ethical Hacker


> A modern, cybersecurity-inspired portfolio website showcasing full-stack development skills and ethical hacking expertise.


## ✨ Features

- 🎨 **Modern Terminal-Inspired Design** - Unique cybersecurity aesthetic
- 🌓 **Dark/Light Mode** - Automatic theme switching with system preference detection
- ⌨️ **Typing Animation** - Dynamic role display (Full Stack Developer, Ethical Hacker, etc.)
- 📱 **Fully Responsive** - Works flawlessly on all devices
- 🔍 **Project Filtering** - Filter projects by category (Web, Backend, Security)
- 📊 **Animated Counters** - Engaging statistics display
- 🔐 **Security-First** - All external links use `noopener` and `noreferrer`
- 📧 **Contact Form** - Ready-to-integrate contact functionality
