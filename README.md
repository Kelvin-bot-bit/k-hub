## 🎨 Features

### Core Features
- ✅ **Terminal-Inspired Design** — Unique cybersecurity aesthetic with terminal blocks
- ✅ **Dark/Light Mode** — Automatic theme switching with system preference detection
- ✅ **Dynamic Typing Animation** — Rotating roles (Full Stack Developer, Ethical Hacker, etc.)
- ✅ **Fully Responsive** — Flawless experience across all devices
- ✅ **Project Filtering** — Filter projects by category (Web, Backend, Security)
- ✅ **Animated Counters** — Engaging statistics with scroll-triggered animation
- ✅ **Security-First** — All external links use `noopener` and `noreferrer`

### Visual Highlights
- 🎯 Interactive hero section with rotating tech icons
- 🌟 Smooth scroll animations and transitions
- 🖥️ Code-like terminal aesthetics
- 🔐 Cybersecurity-inspired design language
- 📱 Mobile-first responsive design
